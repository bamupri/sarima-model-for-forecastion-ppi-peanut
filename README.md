# sarima-model-for-forecastion-ppi-peanut
Analyze the data which contains 'observation_date' and 'WPU01830111' columns, to predict the production price index for the next 24 months using the SARIMA model. Include analysis using ACF, PACF, Seasonal Plot, ADF test, KPSS test, AIC, BIC, HQIC, Ljung-Box, ACF residual, normality, MAE, RMSE, and MAPE, along with relevant plots.
